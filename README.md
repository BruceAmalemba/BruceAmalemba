## AI & Machine Learning
Machine learning pulled me in before the current wave—because it sits where **statistics, optimization, and computation** meet. The pace of AI progress right now is wild, and I’m grateful to be building in this era.

I use modern AI tools as accelerators for research, drafting, and iteration, while keeping the serious work disciplined: problem framing, data quality, evaluation, and accountability. I’m big on helping people adopt AI tools the right way—so they can move faster without losing accuracy or judgment.

## How I Work
- Clarify the decision first (what needs to change, what success looks like)
- Structure the data (definitions, tables, validation rules)
- Build the model/dashboard (clean logic, clear visuals, consistent refresh)
- Deliver with documentation (so the work stays usable after handover)
