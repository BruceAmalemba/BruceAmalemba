## What I Do (for real)
- **Business Intelligence & Dashboards:** Power BI dashboards that answer questions fast (KPIs, trends, breakdowns, drilldowns).
- **Data Cleaning & Structuring:** messy Excel/CSV → clean tables → reliable models (Power Query / pandas / SQL).
- **Analytics & Statistics:** forecasting, experiments, inference, and decision support (R / Python / statsmodels).
- **Software for data workflows:** APIs + systems that keep data moving (Django/REST + Postgres + automation).
